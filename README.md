# 🌐 Portfolio Website

My personal developer portfolio — built with pure HTML, CSS, and JavaScript.
Deployed live on GitHub Pages with automated CI/CD.

🔗 **Live site:** [fidelcoder7.github.io](https://fidelcoder7.github.io)

![Deploy](https://github.com/FidelCoder7/fidelcoder7.github.io/actions/workflows/deploy.yml/badge.svg)

## Features

- Typing effect hero with animated badge
- Project showcase with live GitHub links
- Animated skill bars (Intersection Observer)
- Fully responsive — mobile, tablet, desktop
- Pure HTML/CSS/JS — no frameworks
- Auto-deployed via GitHub Actions on every push

## Stack

HTML5 · CSS3 · Vanilla JavaScript · GitHub Pages · GitHub Actions
